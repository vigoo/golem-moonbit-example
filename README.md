# golem-moonbit-example

[BLOG POST](https://blog.vigoo.dev/posts/moonbit-with-golem/)

Tested with Golem 1.1

Required tools:
- `golem`
- `moon`
- `wit-bindgen`
- `wasm-tools`
- `wit-deps`

To set up dependencies (first time only):

```sh
cd common-deps
wit-deps update
```

To build all components, use:

```sh
golem app build
```
